# Install npm packages
- `npm install`

# Run vite instance
- `npm run dev`

# Pages (Make sure to have nothing else running on port 5173)
- Home: `http://localhost:5173`
- SearchResult: `http://localhost:5173/searchresult`
- Checkout: `http://localhost:5173/checkout`
- Error Example: `http://localhost:5173/errorexample`
